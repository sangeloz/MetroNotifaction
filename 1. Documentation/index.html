Installation

  1. Open the "5. Main Files" Folder
  2. Copy the Static folder to your project.
  3. Add the respective reference to the Font-Awesome.css, MetroNotification.js and MetroNotificationStyle.css

 

Like this:

 

<!-- jQuery Basic Library ( you can use your own version ) -->
<script src="static/js/jquery-1.11.1.min.js"></script>


<!-- Metro Notification Style -->
<link rel="stylesheet" type="text/css" href="static/css/MetroNotificationStyle.min.css">


<!-- Metro Notification JS -->
<script src="static/js/MetroNotification.min.js"></script>


<!-- Font Awesome v4.2 -->
<link rel="stylesheet" type="text/css" href="font-awesome/css/font-awesome.min.css">

 

is very important to understand the following:

  1. I don't use a Custom jQuery Version. If you are using any from 1.7, you can use that instead.
  2. MetroNotification.js, REQUIRES jQuery, so, make sure you place that file after importing jQuery
  3. Font-Awesome 4.2 is included, you can update Font-Awesome by replacing those files.
  4. Font-Awesome is required for BigBoxes and all the "fa" properties.

Basic Call

All the plugins work similar, so I will explain a Basic call. But remember, open tutorial folder so you can get a better understanding and use examples that work in your computer.

 

<!-- Launch a SmallBox on form load -->
<script type="text/javascript">


$(document).ready(function(){ 

    $.smallBox({
        content:"Hi there",
    });

});


</script>

 

That's all, you can add properties to customize the default behavior and aspect.

 

Like this for example:

 

<script type="text/javascript">

$(document).ready(function(){ 

   $.smallBox({
       title:"Your Title",
       content:"Some content",
       icon:"static/img/origami.png",
       color:"#AD0C8D",
   });

});

</script>
That will fire a SmallBox on form load with a custom title, content, icon and color.

Kill the Plugins

If you need to manually close all the plugins:

 

This are the functions that you need to call.

 

<script type="text/javascript">

$.smallBoxKill(); // Close all the SmallBoxes with animation

$.sidePanelKill(); // Close all the SidePanels with animation

$.bigBoxKill(); // Close all the BigBoxes with animation

$.metroLoadingKill(); // Close all the Loadings with animation


</script>

 

Important, if you are using the previous MetroUnLoading(), you should create this function to reduce the impact on your production machine

(OPTIONAL) THIS IS NOT REQUIRED IF YOU START WITH VERSION 7

 
function MetroUnLoading() 
{
   $.metroLoadingKill(); 
}
